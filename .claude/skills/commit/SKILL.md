# コミットメッセージ提案

以下の変更内容を確認して、日本語1行のコミットメッセージを提案してください。

## 現在の変更状況

### git status
!`git status --short`

### ステージ済みの差分
!`git diff --cached`

### 未ステージの差分
!`git diff`

## 要件

- メッセージは**1行・72文字以内**で簡潔に
- **日本語**で記述する（プレフィックスは英字）
- 「何を変更したか」を端的に表現する
- 変更が複数ある場合は最も重要な変更を代表させる
- 下記の **Conventional Commits** プレフィックスを必ず付ける

## プレフィックス一覧（Conventional Commits）

| プレフィックス | 使いどき |
|---|---|
| `feat` | 新機能の追加 |
| `fix` | バグ修正 |
| `docs` | ドキュメントのみの変更 |
| `style` | 動作に影響しない変更（フォーマット・空白・セミコロン等） |
| `refactor` | バグ修正でも機能追加でもないコード変更 |
| `perf` | パフォーマンス改善 |
| `test` | テストの追加・修正 |
| `chore` | ビルドプロセス・補助ツール・設定ファイルの変更 |
| `ci` | CI 設定・スクリプトの変更 |
| `revert` | 以前のコミットの取り消し |

形式: `<type>: <日本語の説明>`
例: `feat: 検索機能にデバウンスを追加`、`fix: ページネーションのオフセット計算を修正`

## 出力形式

提案メッセージを1行だけ出力してください。説明は不要です。
